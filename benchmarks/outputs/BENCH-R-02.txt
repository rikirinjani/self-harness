# BENCH-R-02: Library API Research - Python json.dumps()

## Library: Python `json` module (standard library)
## Use case: Custom JSON serialization

### Key API: json.dumps()

```python
json.dumps(obj, *, skipkeys=False, ensure_ascii=True, check_circular=True,
           allow_nan=True, cls=None, indent=None, separators=None,
           default=None, sort_keys=False, **kw)
```

### Parameters for custom serialization:

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `obj` | object | required | Python object to serialize |
| `default` | callable | None | Function called for non-serializable objects; must return JSON-encodable value or raise TypeError |
| `cls` | JSONEncoder subclass | None | Custom encoder class overriding `default()` |
| `indent` | int/str/None | None | Pretty-print: positive int = spaces, string = literal indent |
| `sort_keys` | bool | False | Sort dict keys alphabetically |
| `skipkeys` | bool | False | Skip non-basic-type keys instead of raising TypeError |
| `ensure_ascii` | bool | True | Escape non-ASCII chars |
| `allow_nan` | bool | True | Allow NaN/Infinity/-Infinity |
| `separators` | tuple | (', ', ': ') | (item_separator, key_separator) |

### Working example:

```python
import json
from datetime import datetime

def custom_serializer(obj):
    if isinstance(obj, datetime):
        return obj.isoformat()
    if isinstance(obj, set):
        return list(obj)
    if isinstance(obj, bytes):
        return obj.hex()
    raise TypeError(f"Type {type(obj)} not serializable")

data = {
    "name": "test",
    "created": datetime.now(),
    "tags": {"python", "json", "serialization"},
    "hash": b"\\x00\\x01\\x02",
    "count": 42
}

result = json.dumps(data, default=custom_serializer, indent=2, sort_keys=True)
print(result)
# Output:
# {
#   "count": 42,
#   "created": "2026-07-08T...",
#   "hash": "000102",
#   "name": "test",
#   "tags": ["json", "python", "serialization"]
# }
```

### Alternative: Custom JSONEncoder subclass

```python
class CustomEncoder(json.JSONEncoder):
    def default(self, obj):
        if isinstance(obj, datetime):
            return obj.isoformat()
        if isinstance(obj, set):
            return list(obj)
        if isinstance(obj, bytes):
            return obj.hex()
        return super().default(obj)

result = json.dumps(data, cls=CustomEncoder, indent=2)
```
